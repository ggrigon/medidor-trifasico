  Função          │ GPIO ESP32  │ Destino              │ Notas

  OLED SCL        │ GPIO22      │ J4.SCL               │ Pull-up 4.7k → 3V3

  SD MISO         │ GPIO19      │ J5.MISO              │ SPI padrão
  SD SCK          │ GPIO18      │ J5.SCK               │ SPI padrão
  SD CS           │ GPIO5       │ J5.CS                │ Chip select
  Buzzer          │ GPIO4       │ BZ1 pin 1            │ Pin 2 → GND
  LED Status      │ GPIO2       │ R3(330Ω) → D1        │ LED verde 3mm
